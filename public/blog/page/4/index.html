Android studio 编译报错 Error:null value in entry: outputDirectory=null

发表于 2017-07-04 | 分类于 IDE

Android studio 编译报错 Error:null value in entry: outputDirectory=null

首先，这是一个bug
然后，请删除根目录下的.gradle文件夹。（注意前面的点，别删错了gradle文件夹）。
再然后clear project，rebuild project。
然后就没事了。

查看apk应用签名

发表于 2017-06-21 | 分类于 android

查看apk的应用签名信息

  • 将apk解压
  • 找到META-INF下的。RSA文件
  • 打开终端，进入.RSA文件所在的目录，命令：keytool -printcert -file CERT.RSA

通过keystore获取应用签名

命令：keytool -list -v -keystore [keystore路径]
